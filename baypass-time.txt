! Title: XLS Timer Bypass & Link Forcer
! Description: Aggressively neutralizes countdown timers and forces hidden download links to appear.
! Expires: 5 days
! Last modified: 2026-07-15
! Homepage: https://github.com/ydwx/home

!
! This filter uses advanced uBlock Origin scriptlet injection
! to sabotage JavaScript timers and remove CSS disguises from download buttons.
! Brave supports this syntax natively when loaded as a custom filter list.

! ====== Timer Annihilation ======
! Disable setTimeout and setInterval entirely if their code contains any of these keywords.
! This prevents the countdown from ever starting, so the download link is immediately available.
*##+js(no-setTimeout-if, /countdown|timer|wait|delay|redirect|cooldown|skip|adblock|throttle/)
*##+js(no-setInterval-if, /countdown|timer|wait|delay|redirect|cooldown|skip|adblock|throttle/)

! ====== Link & Button Revealer ======
! Remove the 'display' attribute from potential download links and buttons,
! forcing them to become visible even if they were hidden by CSS.
*##+js(ra, a[href*="download"], style, display)
*##+js(ra, a[href*="file"], style, display)
*##+js(ra, a[href*="get"], style, display)
*##+js(ra, a[href*="link"], style, display)
*##+js(ra, a[href*="direct"], style, display)
*##+js(ra, a[href*="dl"], style, display)
*##+js(ra, button, style, display)

! Remove HTML 'hidden' and 'aria-hidden' attributes that might hide links.
*##+js(ra, a, hidden)
*##+js(ra, a, aria-hidden)
*##+js(ra, button, hidden)
*##+js(ra, button, aria-hidden)

! Remove classes commonly used to hide elements during a countdown.
*##+js(ra, .hidden, style, display)
*##+js(ra, .invisible, style, display)
*##+js(ra, [class*="hide"], style, display)
*##+js(ra, [class*="d-none"], style, display)

! ====== Generic Overlay Killer ======
! Many sites place a transparent overlay that blocks clicks.
! This rule removes any div that has a high z-index and is likely a blocker.
*##div[style*="z-index: 99"]:has(~ a[href*="download"])
*##div[style*="z-index: 100"]:has(~ a[href*="download"])
*##div[class*="overlay"]:has(~ a[href*="download"])
*##div[id*="overlay"]:has(~ a[href*="download"])
